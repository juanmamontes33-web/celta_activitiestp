# English Class Materials

Interactive HTML slides, student activities and teacher scripts for English lessons,
hosted on GitHub Pages.

**▶ Live hub:** https://juanmamontes33-web.github.io/celta_activitiestp/

The homepage (`index.html`) is the hub — every lesson is grouped by course and lesson,
with links to Slides, Activities and Extras.

## Contents

- **English from Zero** — a 22-lesson course covering the tenses, comparatives,
  superlatives and reading/listening units.
- **More Grammar & Skills** — standalone lessons and shared resources.
- **CELTA — Teaching Practice** — assessed lessons TP2–TP8 (slides, activities, teacher scripts).

## File naming

| Prefix | Meaning |
|---|---|
| `Class_Slides_*` | Presentation slides for a lesson |
| `Activity_*` / `Student_Activity_*` | Student practice activity |
| `Teacher_Script_*` | Teacher's lesson script (CELTA) |
| `Discovery_Tasks_*` | Guided discovery activity |
| `TP#` | CELTA Teaching Practice number |

## /archive

Older or duplicate versions of files are kept in `archive/` so the main folder stays clean.
They are not linked from the hub.
