How do we build it? — Form

used to + base verb. Watch the negative and question carefully!

Form 🔧
➕ AFFIRMATIVE
I used to play.
used to + base verb
➖ NEGATIVE
I didn't use to play.
didn't use to + base verb
❓ QUESTION
Did you use to play?
Did + use to + base verb
⚠️ THE GOLDEN RULE — the disappearing "d"
➕ used to
→ after didn't / Did, drop the d:
➖❓ use to (NOT used to)
Why? Did / didn't already shows the past — so we don't need the -d twice. Same idea as "Did you go?" (not Did you went?).
Pronunciation · listen & repeat
It sounds like "YOO-stuh" 🔊
Surprise! We don't say "use-D too". The d almost disappears and "to" becomes weak.
used to / use to
→ /ˈjuːstə/
"YOO-stuh" — one smooth sound, weak ending
⚠️ Careful!
/juːzd tuː/ ❌
Don't say each part heavily — blend them!
DRILL — repeat after me (slowly, then fast):
🔁 "I used to play."   /aɪ ˈjuːstə pleɪ/
🔁 "Did you use to live here?"   /dɪdʒə ˈjuːstə lɪv/
🔁 "I didn't use to like it."   /aɪ ˈdɪdnt ˈjuːstə/
