FORM
How it works
The action moves to the front 🔄
Active — who does it
A local family runs the café.
Passive — what happens
The café is run (by a local family).
subject + be + past participle (+ by + agent)
The tense lives in be: is run · was run · has been run · will be run · can be run.

Open-class feedback
Answers — and why 💬
1 Passive (is chosen) · 2 Active (rented — no be) · 3 Passive (were picked up) · 4 Passive (was built)
When do we choose the passive?
When the doer is unknown, obvious or unimportant — or to keep the focus on the place/action.
"Breakfast is served at eight." — we don't care who serves it.

LANGUAGE
Language feedback · MFP
Get it right 🛠️
Common errors
is ran → is run · has chosen → has been chosen · should cleaned → should be cleaned
Don't forget been
Perfect & modal passives need it:
has been run · can be learned · will be served
Pronunciation
Weak is/was/been → /ɪz/ /wəz/ /bɪn/. "The café's run by…" (contracted is).
Drop the agent?
Keep by… only if it adds info. "was built by builders" → just was built.
